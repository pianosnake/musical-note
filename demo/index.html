<musical-note> demo

Define note by name and octave in the value attribute.

Define using a MIDI note number. Middle C is 48. C# above that is 49.

Sharps are used by default, but the flat attribute will ensure the note is drawn using a flat.

Create a taller container by specifying a height attribute.

Update the value of a note using the setValue method. If the second parameter to setValue is true the note will be drawn using a flat.
