# <musical-note>

A Polymer element for a musical note in treble clef.

## Usage

```html
<musical-note value="c"></musical-note>
<musical-note value="c#"></musical-note>
<musical-note value="c#5"></musical-note>
<musical-note value="eb5></musical-note>
```
